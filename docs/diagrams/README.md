# Диаграммы проекта

PNG-файлы сгенерированы скриптом `tools/generate_diagrams.php`.

- `use-case.png` - диаграмма вариантов использования.
- `activity.png` - диаграмма деятельности.
- `sequence.png` - диаграмма последовательности.

Чтобы пересоздать картинки:

```powershell
C:\OSPanel\modules\php\PHP_8.1\php.exe tools\generate_diagrams.php
```
