# Рекомендации по оптимизации изображений для платформы "НаСтарте"

## Общие рекомендации

### 1. Форматы изображений
- **WebP/AVIF** — используйте современные форматы для лучшей компрессии
- **JPEG** — для фотографий (качество 75-85%)
- **PNG** — только для изображений с прозрачностью
- **SVG** — для иконок и логотипов

### 2. Размеры изображений

#### Аватары пользователей
- Оптимальный размер: 200x200px
- Максимальный размер файла: 50KB
- Формат: WebP или JPEG

#### Изображения для карусели
- Рекомендуемый размер: 1200x800px
- Максимальный размер файла: 150KB
- Формат: WebP

#### Превью соревнований
- Рекомендуемый размер: 800x600px
- Максимальный размер файла: 100KB
- Формат: WebP или JPEG

### 3. Сжатие изображений

Перед загрузкой изображений используйте инструменты сжатия:
- **TinyPNG** (https://tinypng.com/)
- **Squoosh** (https://squoosh.app/)
- **ImageOptim** (для Mac)
- **FileOptimizer** (для Windows)

### 4. Lazy Loading

Все изображения ниже первого экрана должны использовать lazy loading:
```html
<img src="image.jpg" alt="Описание" loading="lazy" />
```

В React/Next.js:
```tsx
<Image
  src="/image.jpg"
  alt="Описание"
  loading="lazy"
  width={800}
  height={600}
/>
```

### 5. Атрибуты доступности

Всегда указывайте:
- `alt` — текстовое описание изображения
- `title` — подсказка при наведении
- `width` и `height` — для предотвращения layout shift

### 6. Кэширование

Настройте кэширование изображений на сервере:
```
Cache-Control: public, max-age=31536000
```

### 7. CDN для изображений

Рекомендуется использовать CDN для доставки изображений:
- Cloudflare Images
- Imgix
- Cloudinary
- AWS CloudFront

## Примеры оптимизации

### До оптимизации
```html
<img src="photo.jpg" alt="фото">
```

### После оптимизации
```html
<img
  src="photo-optimized.webp"
  alt="Спортивные соревнования по бегу на стадионе"
  title="Соревнования по бегу 2026"
  width="800"
  height="600"
  loading="lazy"
  decoding="async"
/>
```

## Инструменты для анализа

- **Google PageSpeed Insights** — анализ производительности
- **Lighthouse** — аудит качества веб-приложений
- **WebPageTest** — тестирование скорости загрузки
- **Screaming Frog** — поиск битых изображений
